Como o modelo calcula

Probabilidade do bilhete = produto das probabilidades restantes de cada seleção × ajuste de correlação × ajuste de perfil. Seleções já cumpridas viram 100%; seleções eliminadas viram 0%; fases detectadas pela API encurtam o caminho restante.
